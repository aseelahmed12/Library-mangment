# نظام إدارة مكتبة (PHP + MySQL)

## المتطلبات المنفذة
- إضافة/عرض/بحث/تعديل/حذف كتب.
- واجهة عربية بسيطة قابلة للاستبدال بقالب جاهز.
- ربط كامل بقاعدة MySQL باستخدام PDO وتهيئة الترميز UTF-8.
- العمل على خادم محلي XAMPP.

## التشغيل السريع
1) شغّل XAMPP (Apache + MySQL).
2) افتح phpMyAdmin وأنشئ قاعدة البيانات عبر استيراد الملف: `library_db.sql`.
3) عدّل بيانات الاتصال داخل `config.php` إذا لزم.
4) انسخ المجلد كاملاً إلى `htdocs` مثل: `C:\xampp\htdocs\library_project_ar`.
5) افتح في المتصفح: `http://localhost/library_project_ar/`.

## ملفات رئيسية
- `index.php` — الصفحة الرئيسية.
- `books_list.php` — عرض + بحث + روابط التعديل/الحذف.
- `book_add.php` — إضافة كتاب + رفع غلاف.
- `book_edit.php` — تعديل بيانات الكتاب.
- `book_delete.php` — حذف كتاب.
- `includes/header.php`, `includes/footer.php` — القالب.
- `includes/upload_helper.php` — دالة رفع الغلاف.
- `assets/uploads/` — مكان حفظ الصور.
- `library_db.sql` — ملف قاعدة البيانات للتسليم.
